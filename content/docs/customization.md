## Layout Options

### Card Styles

- **Solid** - Opaque background cards
- **Glass** - Glassmorphism with blur effect
- **Outline** - Border-only cards

### Border Radius

- **Sharp** - 8px corners
- **Rounded** - 16px corners (default)
- **Pill** - Fully rounded corners

### Spacing

Adjust padding and gaps between elements:

- Link spacing: 8px - 24px
- Card padding: 12px - 32px
- Section gaps: 16px - 48px

## Pro Features

### Custom CSS (Pro+)

Write your own CSS to override any styles:

```css
/* Example: Custom link hover effect */
.link-card:hover {
  transform: scale(1.05) rotate(1deg);
  box-shadow: 0 20px 40px rgba(0, 0, 0, 0.3);
}
```

**Limitations:**
- Max 10,000 characters
- No `@import` or external resources
- No `position: fixed` (security)
- No JavaScript expressions

## Best Practices

1. **Contrast** - Ensure text is readable against your background
2. **Consistency** - Use complementary colors and fonts
3. **Performance** - Avoid heavy video backgrounds on mobile
4. **Accessibility** - Test with screen readers and keyboard navigation
