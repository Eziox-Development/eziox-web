## Free Tier (Eziox Core)

Everything you need to create a beautiful bio page:

- **Unlimited links** and short links
- **All 31+ themes** and color schemes
- **All backgrounds** (solid, gradient, image, video, animated)
- **All animations** and effects
- **Spotify, SoundCloud & YouTube** embeds
- **Social media icons** and integrations
- **Profile picture & banner** customization
- **Full analytics dashboard** (30-day retention)
- **Mobile-optimized** responsive design

**Limitations:**
- Subtle "Powered by Eziox" branding
- No custom CSS or fonts
- No analytics export

## Pro Tier

Essential power features for customization enthusiasts:

### Custom CSS

Write your own styles to override any element:

```css
/* Make links glow on hover */
.link-card:hover {
  box-shadow: 0 0 30px var(--primary);
}
```

### Custom Fonts

Upload up to 4 fonts from Google Fonts or custom URLs.

### Remove Branding

Hide the "Powered by Eziox" footer link.

### Profile Backups

Automatic backups with restore functionality.

### Export Analytics

Download your data as CSV or JSON.

### Pro Badge

Show off your support with a Pro badge on your profile.

## Creator Tier

Professional toolkit for serious creators:

### Custom Domain

Use your own domain (e.g., `links.yourdomain.com`):

1. Add your domain in Dashboard → Creator
2. Configure DNS (CNAME to `cname.eziox.link`)
3. Wait for SSL certificate (automatic)

### Password-Protected Links

Restrict access to specific links with a password.

### Link Expiration

Set automatic expiration dates for time-sensitive content.

### Email Collection

Collect visitor emails with built-in forms:

- Customizable form design
- Export subscriber list
- GDPR-compliant opt-in

### Custom Open Graph

Control how your page appears when shared:

- Custom title and description
- Custom preview image
- Platform-specific settings

### Priority Support

Get faster response times via email.

### Early Access

Be the first to try new features.

### Creator Badge

Exclusive Creator badge on your profile.
