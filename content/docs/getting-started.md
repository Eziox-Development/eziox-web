# Getting Started with Eziox

Welcome to Eziox! This guide will help you create your personalized bio page in just a few minutes.

## Creating Your Account

1. Visit [eziox.link/sign-up](/sign-up) to create your account
2. Enter your email address and choose a secure password
3. Pick a unique username - this will be your bio page URL (`eziox.link/yourusername`)
4. Verify your email address by clicking the link we send you

## Setting Up Your Profile

Once you're logged in, head to your [Dashboard](/profile) to customize your profile:

### Basic Information

- **Display Name** - Your public name shown on your bio page
- **Bio** - A short description about yourself (max 160 characters)
- **Avatar** - Upload a profile picture (recommended: 400x400px)
- **Banner** - Add a header image (recommended: 1200x400px)

### Social Links

Connect your social media accounts:

- Twitter/X
- Instagram
- YouTube
- TikTok
- Twitch
- Discord
- GitHub
- LinkedIn

## Adding Links

Your bio page can contain unlimited links. To add a link:

1. Go to **Dashboard → Links**
2. Click **Add Link**
3. Enter the URL and a title
4. Optionally add an icon and description
5. Drag to reorder your links

### Link Features

- **Thumbnails** - Add custom images to your links
- **Scheduling** - Set start/end dates for time-limited links
- **Analytics** - Track clicks on each link
- **Featured** - Highlight important links with special styles

## Choosing a Theme

Eziox offers 31+ themes across 8 categories:

| Category | Themes |
|----------|--------|
| General | eziox-default, obsidian, midnight, ember |
| Gamer | neon-green, rgb-fusion, cyberpunk |
| VTuber | kawaii-pink, pastel-dream, anime-night |
| Developer | terminal, github-dark, vscode |
| Streamer | twitch, youtube, kick |
| Artist | canvas, watercolor, gallery |
| Minimal | minimal-dark, minimal-light, aurora |
| Premium | ocean-depths, forest-night, neon-tokyo |

To change your theme:

1. Go to **Dashboard → Customization**
2. Browse the theme gallery
3. Click on a theme to preview it
4. Click **Apply** to save
